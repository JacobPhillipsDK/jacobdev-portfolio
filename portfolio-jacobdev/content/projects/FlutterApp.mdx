This project is a Flutter-based Android application developed to provide stroke patients with a chatbot that offers personalized recovery advice and general health improvement tips. It features a robust notification system and backend integration using Firebase, ensuring seamless communication and user interaction.

### Project Highlights:

- **Chatbot Development:** Created a chatbot knowledgeable in stroke recovery and general health improvement, providing users with timely and accurate advice.
- **Notification System:** Integrated a custom notification system with actionable buttons to engage users and provide real-time updates.
- **UI/UX Design:** Led the design and implementation of an intuitive and user-friendly interface for stroke patients.
- **Backend & Database Management:** Used Firebase for app backend services, including real-time database management, user authentication, and notifications.
- **Python Automation:** Developed Python scripts to send personalized notifications to users, enhancing the user experience with timely reminders.
