# Welcome to My Portfolio

Hi, I'm Jacob Phillips, a passionate software developer with a keen interest in virtual reality, human-computer interaction (HCI), and full-stack web development. This is a collection of my work, showcasing projects that reflect my skills and research in these fields.

## Projects

In my portfolio, you'll find a variety of projects ranging from cutting-edge research in virtual reality to practical software solutions. Below are some highlights:

### [Visualization of Walking Routes in Aalborg Using A* and Mapbox on OSM Data 2024](https://jacobdev.dk/projects/WalkingRoutes)
This project utilizes the A* algorithm with OpenStreetMap (OSM) data to visualize optimal walking routes in Aalborg, Denmark, through interactive maps powered by Mapbox. It also features a custom street view experience for virtual route exploration.

### [Discount Tracking Application](https://jacobdev.dk//projects/Discount-Finder)
An application designed to help users track price histories of various products, giving insights into when the best time to buy might be. This project combines data scraping with real-time analytics.
